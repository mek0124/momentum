## How To Use

- To use the executable file:
  - visit the [releases](https://github.com/mek0124/task-manager/releases/)
  - locate the latest version for your operating system
  - download the executable for your operating system
  - click to run the executable
- To compile from source:
  - Clone the repo
    - `git clone https://github.com/mek0124/task-manager.git`
  - cd into the project
    - `cd task-manager`
  - run the install command to install all necessary dependencies
    - `pnpm run custom:install`
  - run the project
    - Here you can either run `pnpm build` to build the project
      - go into the src-tauri/target/release and run the task-manager app image
    - or
    - run `pnpm run start`
